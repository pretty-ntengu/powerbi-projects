# Car Sales Dashboard

## Project Overview
This Power BI dashboard was created as part of a guided training session with Exodus Experts. The project involved following a step-by-step demonstration to build a car sales dashboard, with the goal of learning Power BI reporting techniques and best practices.

## Learning Objectives
- Understand Power BI dashboard structure
- Practice data cleaning and transformation using Power Query
- Learn how to create DAX measures
- Build interactive visuals using slicers and filters
- Interpret basic sales insights from data

## Data
- Source: Training dataset provided during Exodus Experts class
- Data Type: Car sales transactions
- Key Fields: Date, Brand, Model, Region, Units Sold, Revenue

## What Was Done
- Followed instructor-led Power BI demonstrations
- Recreated the dashboard independently using recorded training videos
- Applied Power Query steps shown in class
- Implemented DAX measures as taught
- Built visuals and slicers according to the lesson

## Tools Used
- Power BI
- Excel

## Skills Practiced
- Power Query (data cleaning & transformation)
- DAX basics
- Data modeling fundamentals
- Interactive dashboard creation
- Report navigation and filtering

## Key Takeaways
- Gained hands-on experience building Power BI dashboards
- Improved understanding of DAX and Power Query
- Learned how to structure clear, user-friendly reports
